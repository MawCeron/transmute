# Data

Data conversions use the bit as the base unit. All other units are expressed as multiples of bits.

## Bits and Bytes

One byte equals 8 bits. This distinction matters: network speeds are typically advertised in bits per second, while storage capacity is quoted in bytes. A gigabit (gbit) is 10⁹ bits; a gigabyte (gb) is 8×10⁹ bits (8 gigabits).

## SI Decimal Prefixes (Powers of 10)

The SI prefixes — kilo, mega, giga, tera, peta, exa — each represent a factor of 1000. These are used by hard drive manufacturers and network equipment vendors.

| Unit | Symbol | Value |
|------|--------|-------|
| Kilobyte | kb | 8,000 bits |
| Megabyte | mb | 8,000,000 bits |
| Gigabyte | gb | 8,000,000,000 bits |
| Terabyte | tb | 8×10¹² bits |

## IEC Binary Prefixes (Powers of 2)

The IEC 80000-13 standard introduced binary prefixes to avoid ambiguity: kibi (1024), mebi (1024²), gibi (1024³), and so on. These are used by operating systems when reporting RAM and file sizes.

| Unit | Symbol | Value |
|------|--------|-------|
| Kibibyte | kib | 8,192 bits |
| Mebibyte | mib | 8,388,608 bits |
| Gibibyte | gib | 8,589,934,592 bits |

## Legacy Units

The nibble (4 bits) and word (16 bits) persist in low-level programming documentation. A double word (dword, 32 bits) and quad word (qword, 64 bits) map to standard CPU register widths.

Early personal computers stored data on floppy disks holding about 1.44 MB (approximately 11.8 million bits) — a nostalgic reference that puts modern storage into perspective when you consider that a single smartphone photo now exceeds the entire capacity of a box of floppies.
